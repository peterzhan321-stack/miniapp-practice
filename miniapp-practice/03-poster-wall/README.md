# 校园活动海报墙小程序

一个基于微信小程序开发的校园活动海报墙应用，采用瀑布流布局展示各类校园活动，支持搜索、发布、收藏、点赞等功能。

## 项目简介

本小程序旨在为校园社团提供一个活动展示平台，学生可以浏览各类活动海报、搜索感兴趣的活动、发布新活动，并对喜欢的活动进行收藏和点赞。

## 功能特性

### 核心功能

| 功能 | 描述 |
|------|------|
| 🔍 活动搜索 | 支持按活动名称实时搜索 |
| 📱 瀑布流展示 | 双列瀑布流布局，类似小红书的视觉体验 |
| 📝 活动发布 | 支持图片上传、分类选择、信息填写 |
| ⭐ 活动收藏 | 详情页可收藏，本地存储保持状态 |
| ❤️ 活动点赞 | 心形点赞按钮，支持取消点赞 |
| 📖 活动详情 | 展示完整活动信息和介绍 |

### 技术特性

- **图片懒加载**：使用 `lazy-load` 优化大量图片加载性能
- **组件化开发**：海报卡片抽离为独立组件，便于复用和维护
- **本地数据持久化**：使用 `wx.setStorageSync` 保存收藏和点赞状态
- **响应式设计**：Flex 布局适配不同屏幕尺寸

## 页面说明

### 首页（pages/index/index）

首页是小程序的主页面，展示所有活动的瀑布流列表。

**主要功能**：
- 顶部搜索栏：实时搜索活动名称
- 瀑布流布局：左右两列展示活动卡片
- 悬浮发布按钮：右下角圆形"+"按钮，点击跳转到发布页
- 下拉刷新：支持下拉刷新获取最新数据
- 上拉加载：滚动到底部自动加载更多

**技术要点**：
- 使用 `scroll-view` 实现滚动容器
- 瀑布流算法：按索引奇偶分配到左右列
- 自定义组件 `poster-card` 展示活动卡片

### 详情页（pages/detail/detail）

详情页展示单个活动的完整信息。

**主要功能**：
- 活动海报大图展示
- 完整活动信息（标题、日期、地点、分类、社团）
- 收藏功能：点击收藏按钮切换收藏状态
- 分享功能：支持分享给微信好友

**技术要点**：
- 使用 `wx.getStorageSync` 读取和保存收藏状态
- 收藏数据存储在本地，刷新后状态保持

### 发布页（pages/publish/publish）

发布页用于发布新的活动。

**主要功能**：
- 图片选择：使用 `wx.chooseMedia` 选择图片
- 标题输入：活动标题文本输入
- 分类选择：学术、体育、艺术三选一
- 地点输入：活动地点填写
- 日期选择：活动日期选择器
- 社团名称：发起社团填写
- 表单验证：所有字段必填才能提交

**技术要点**：
- 使用 `picker` 组件实现分类和日期选择
- 表单验证实时检查
- 发布后数据通过全局变量传递给首页

## 组件说明

### poster-card 组件

海报卡片组件是小程序的核心展示组件。

**属性（Properties）**：

| 属性名 | 类型 | 说明 |
|--------|------|------|

- 图片懒加载：使用 `lazy-load` 属性优化性能
- 加载占位：图片加载前显示旋转动画
- 淡入效果：图片加载完成后渐变显示
- 点赞功能：心形按钮，支持本地存储

## 懒加载实现

### 什么是懒加载？

懒加载（Lazy Load）是一种延迟加载技术，只有当图片即将进入可视区域时才加载，而不是页面初始化时一次性加载所有图片。

### 实现方式

在小程序中，通过 `image` 组件的 `lazy-load` 属性实现：

```html
<image 
  src="{{poster.imageUrl}}" 
  lazy-load
  bindload="onImageLoad"
/>
```

### 工作原理

1. **初始状态**：图片不加载，只显示占位区域
2. **滚动检测**：小程序监听 scroll-view 滚动事件
3. **接近可视区**：当图片距离可视区一定距离（通常 500px）时触发加载
4. **加载完成**：图片加载成功后显示，并触发 `bindload` 事件

### 优化效果

- 减少首屏加载时间
- 节省用户流量
- 降低内存占用
- 提升页面渲染性能

## 数据存储

### 本地存储键值说明

| 键名 | 用途 | 数据格式 |
|------|------|----------|
| favorites | 存储收藏的活动列表 | Array<{id, title, timestamp}> |
| likes | 存储点赞的活动列表 | Array<{id, title, timestamp}> |

### 使用示例

```javascript
// 读取收藏列表
const favorites = wx.getStorageSync('favorites') || [];

// 保存收藏列表
wx.setStorageSync('favorites', favorites);

// 清空收藏
wx.removeStorageSync('favorites');
```

## 运行项目

### 环境要求

- 微信开发者工具（最新版）
- 微信基础库版本 3.0.0+

### 运行步骤

1. 克隆项目到本地
2. 打开微信开发者工具
3. 选择"导入项目". 选择项目根目录
5. 填写 AppID（测试可使用测试号）
6. 点击"确定"运行
